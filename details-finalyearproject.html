Project information

  • Category: Project
  • Organization: EPHEC

Project description: For my last year in college, we have to do a project. My project was about making a website plateform to simplify the use of Proxmox VE. My teacher had a cluster of servers with Proxmox VE installed on them. Proxmox VE was used to manage virtual machines and containers, the problem of my teacher was he needed to create virtual machines for each students and it was a long and boring process. So I decided to make a website plateform to simplify the use of Proxmox VE. The website was made with NextJS and Python with flask.
